# Photography MERN Starter

This starter includes:

- Frontend: React + Vite + Tailwind
- Backend: Node + Express + MongoDB (Mongoose)
- Authentication (JWT), Booking APIs, basic models

## How to run

### Backend
1. cd backend
2. cp .env.example .env and set values
3. npm install
4. npm run dev

### Frontend
1. cd frontend
2. npm install
3. npm run dev

Frontend expects backend API at `http://localhost:5000/api` by default.
